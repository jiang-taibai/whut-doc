# LabelImg

## 简介

官网：[https://github.com/HumanSignal/labelImg](https://github.com/HumanSignal/labelImg)

LabelImg是一个用于图像标注的开源工具，它可以帮助我们快速标注图像中的目标，生成VOC、YOLO格式的标注文件。

## 安装启动

```bash
pip install PyQt5
pip install pyqt5-tools
pip install lxml
pip install labelimg
# 直接在命令行中输入labelimg启动
labelimg
```
